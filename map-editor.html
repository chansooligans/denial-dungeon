Map editor — level1

Click object to select. Drag to move (must land on floor).
F flip horizontal. 0 reset flip. Del remove.
+ / − / [ / ] resize selected (0.4×–2.5×). 1 reset size.
Cmd/Ctrl + scroll or the toolbar buttons to zoom the map.
Click empty floor → palette opens with every available sprite. Cells marked · are inactive (no glyph mapping) — placing them emits a tileMeta.sprite override.
Output is two blocks. tileMeta carries sprite, size, and flip overrides; tileOverrides only emits when something was deleted. Paste both into src/content/maps/level1.ts.
