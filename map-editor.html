Map editor — level1

Click object to select. Drag to move (must land on floor).
Q/E rotate ±90°. R / Shift+R rotate ±15°. F flip horizontal. 0 reset. Del remove.
Click empty floor → palette → choose glyph to add.
Output is two blocks: tileMeta for rotations/flips, tileOverrides for added/moved/removed objects. Paste both into src/content/maps/level1.ts.
